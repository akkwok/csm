Out-of-band Management (OOB)

Selected ASUS motherboards feature out-of-band management (OOB) with a BMC module, DASH LAN port, IPMI expansion card or Intel® vPro® technology design. This enables remote device access and status recovery for unresponsive devices, which is crucially important for troubleshooting so that IT administrators can fix the problem remotely.

Enterprise-grade Security

ASUS Control Center Express allows you to securely control unauthorized system access by providing remote disablement of the Registry Editor, blocking USB storage for all devices and providing automatic reboots via watchdog programs. A software blacklist is also available to prevent running of high-risk applications for IT security.

Key Benefits

WHQL Reseller Program

WHQL (Windows Hardware Quality Lab) is Microsoft’s testing process and logo program. It provides hardware manufacturers and end users with reassurances that WHQL-certified systems and peripherals will work with the Windows operating system.

How to Apply for ASUS Reseller Submission Program

The ASUS Reseller Submission Program allows customers who purchase WHQL-certified ASUS motherboards to add their own system logo after repackaging. This program grants resellers permission to use the WHQL certification, saving time and money on testing and validation.

  • Step 1: Apply

    Apply to become an authorized user under Windows Dev Center

  • Step 2: Authorize

    Request permission to use WHQL certification

  • Step 3: Submit

    Log in Windows Dev Center and accept submission
